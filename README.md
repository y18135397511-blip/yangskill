# Healing Companion - 疗愈助手

AI 疗愈助手，帮助 20 岁左右的大学生缓解日常焦虑和空虚感。

## 功能

- AI 引导式情绪签到
- 微行动建议与反馈
- 周回顾情绪曲线可视化
- localStorage 数据持久化

## 技术栈

- Next.js 16
- TypeScript
- Tailwind CSS

## 本地运行

```bash
npm install
npm run dev
```

## 部署

部署到 Vercel：
```bash
npx vercel
```

## 配置

在 `.env.local` 中配置 API Key：

```
KIMI_API_KEY=your_api_key_here
```
