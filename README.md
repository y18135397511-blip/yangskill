# yangskill

AI 产品经理技能库 - 存放各种帮助 AI PM 提升效率的 Skills

## 什么是 Skill？

Skill 是专门为 AI 产品经理打造的提示词和工作流模板，可以帮助你快速完成特定任务。

## 现有 Skills

### 🎯 vibe-coding-prd
帮助 AI 产品经理用对话式引导快速生成 vibe coding 项目的 PRD。

**功能**：
- 5 个问题快速澄清需求
- 自动推荐技术方案
- 生成一页纸 PRD
- 质量自检清单

**适用场景**：
- "我想做一个 XX"
- "帮我做个 YYY 工具"
- "帮我写个 PRD"

**查看详情**：[vibe-coding-prd/SKILL.md](./vibe-coding-prd/SKILL.md)

## 如何使用

1. 克隆仓库到本地
2. 根据需要选择对应的 Skill
3. 按照 Skill 中的指引操作
